# Sistem Rental Kamera (Camera Rental System)

A desktop application for managing camera rental transactions, built with Java Swing for the user interface and MySQL for database management. Developed as the final project for the Object-Oriented Programming course.

## Features
- Camera inventory management (add, edit, delete data)
- Rental transaction processing
- Late return penalty calculation
- Transaction report generation

## Technologies Used
- Java (Swing)
- MySQL
- IntelliJ IDEA

## Project Background
This application applies core Object-Oriented Programming principles such as encapsulation, inheritance, and polymorphism to solve a real business case: managing a camera rental service.
